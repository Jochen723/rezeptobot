Introduction

Grill is Responsive Recipes & Food Blog Template built with Bootstrap v4, HTML5 and CSS3.

This template focused on recipe and food blogs. It helps you to present your recipes in a most attractive way. Grill suits for Food, Recipes Blogs, and Cooking websites.

Grill is fully responsive, so your creation is guaranteed to look great and work perfectly on any device.

Well Organized and valid Code.

This Documentation provide you a clear guideline for Grill to use and customize template. If you facing any problem, please don't forget to ask any question or contact with us for support. Thank You for Purchasing our Product.

LESS, CSS Files & Structure

We are using LESS CSS pre-processor to develop this templates.

In folder src (source) you will found folder with name "less". This folder contain 3 file with extention .less. i.e. base.less, responsive.less, styles.less

base.less — Less css basic settings.

responsive.less — Less css responsive settings.

styles.less — The main css templates.



Result of process generating less file :

styles.css— compact version

styles.min.css— minified version



Every part of LESS File is seperated with Comments.
