Grill is Responsive Recipes & Food Blog Template built with Bootstrap v4, HTML5 and CSS3.

This template focused on recipe and food blogs. It helps you to present your recipes in a most attractive way.

Grill suits for Food, Recipes Blogs, and Cooking websites.

Grill is fully responsive, so your creation is guaranteed to look great and work perfectly on any device.

grill has support for Gulp so you can create your custom template very easily automated.

In the gulpfile.js there are 4 automatic gulp task :

1. HTML - to clean and format html code.
2. Less - to live compile LESS files.
3. Javascript - to compile your javascript files.
4. Image - to compress image size.


In every template there are 3 folder:
1. dist - Main final template.  
2. doc - Template documentation
3. src - Source code template (html, Less, images and js)
